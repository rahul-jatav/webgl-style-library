# WebGL Style Library

Three live WebGL rendering styles — Mercuria, Abyss, Brushfield — with a copy-paste code generator.

Single-file static site. No build step.

## Local dev
```
npx http-server . -p 5180 -c-1 --cors
```
Open http://localhost:5180/

## Deploy
Auto-deploys to Vercel on push to `main`.
